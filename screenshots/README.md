# Screenshots

This directory contains screenshots showcasing MetaImGUI's features and UI.

## Required Screenshots

To complete the documentation, capture the following screenshots:

1. **main-window.png** - Main application window with default theme
2. **themes.png** - Side-by-side comparison of Dark, Light, Classic, and Modern themes
3. **dialogs.png** - Examples of dialog system (message boxes, input, progress)
4. **about-dialog.png** - About dialog showing version information
5. **menu-system.png** - Menu bar with dropdown menus visible

## Screenshot Guidelines

- Use PNG format for best quality
- Capture at reasonable resolution (not too large, not too small)
- Ensure clean, uncluttered background
- Show realistic but non-sensitive example data
- Consistent window sizing across screenshots

## Optional Screenshots

- update-notification.png - Update checker showing available update
- multi-platform.png - Application running on Linux, Windows, and macOS
- localization.png - Application in different languages

## Usage in Documentation

These screenshots are referenced in the main README.md to help users understand what the template provides.
